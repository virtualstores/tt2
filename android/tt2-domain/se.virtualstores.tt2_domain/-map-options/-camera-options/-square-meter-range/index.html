data class SquareMeterRange(from: Double, to: Double, zoomLevel: Double)
Content copied to clipboard

Defines a range of the Venues total area in square meter. Used by different camera modes to set a zoom level that is acceptable for the size of the venue. A smaller venue typically needs a higher zoom level for some camera modes to give a better user experience.

Parameters

from

defines the lower range square meter value for which this zoom level will be used

to

defines the upper range square meter value for which this zoom level will be used

zoomLevel

defines the zoom level that is to be used for Venues with an area that is within the from and to range.
