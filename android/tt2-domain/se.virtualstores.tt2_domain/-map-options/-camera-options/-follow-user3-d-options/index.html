Settings for the follow user 3D camera mode. Based of the Venues total area in square meter. A smaller venue typically needs a higher zoom level for this camera mode to give a better user experience.

Parameters

zoomLevels

defines the zoom levels that is to be used based on the total area of the Venues.

defaultZoomLevel

defines the default zoom level that is to be used for Venues with an area that is not within the zoom level ranges.
