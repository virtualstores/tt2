MapController

interface MapController

Map controller contains tools for controlling the visible map. It is recommended to use the MapController with the TT2 sdk positioning system.

abstract val camera: CameraController

Camera controller contains methods for controlling the behaviour of the camera with different CameraController.CameraModes. For example, showing the hole map or following the person with a certain zoom level.

Location controller contains user mark related methods. It is mainly used for updating the users position on the map with LocationController.updateUserLocation, with positions given by the TT2 system.
