This MapMark class has many built in features and has a number of display options. It can show either: - an image from url or a resource. - or a short text field. It also has a spot for a decoration icon in the top right corner as well as quantity label in the top left.

Needs to be unique.

The location of the mark.

floorLevelId

data

(Optional) Generic data type where you can put any data that you want to access.

isDeletable

Danger: Creates a permanent mark. App needs to be restarted to remove it.

triggerRadius

(Optional) if set the MarkerController will trigger events when user enters/exits the radius of this mark.

isClusterable

Determines if the mark should be clustered if it overlaps with another mark, i.e if you zoom out the map to that point.

isVisible

if the mark is visible or not.

isFocused

if the mark should is focused or not. Focused marks gets scaled up to a bigger size.

decoration

(Optional) Resource id for a decoration icon, will render if set.

quantity

(Optional) String value for a quantity label, will render if set.

imageURL

(Optional) String value url for loading an imageURL.

imageSrcId

(Optional) Resource id for displaying an iconl.

text

(Optional) String value for displaying a short text.

quantityBackgroundColor

(@ColorInt) sets the background color of the quantity label.

quantityFont

(Typeface) (optional) change the typeface of the quantity label.

quantityTextColor

(@ColorInt) sets the text color of the quantity label.

opacity

(optional) sets the transparency on the rendered mark, this will override MapMark.opacity in MapOptions
