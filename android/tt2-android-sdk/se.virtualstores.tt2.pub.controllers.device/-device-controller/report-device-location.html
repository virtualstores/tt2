reportDeviceLocation

abstract fun reportDeviceLocation(state: DevicePosition.DevicePositionState = DevicePosition.DevicePositionState.ACTIVE, deviceId: String, description: String? = null, tags: Map<String, String>? = null, position: CoordinateF? = null, lngLat: List<Double>? = null, callback: (Throwable?) -> Unit? = null)

Reports the current location of the device to the CMS where you can visually track the device.

Parameters

state

the state of the device, defaults to ACTIVE if not set otherwise.

deviceId

device id that will be displayed on the CMS.

description

optional short string where you can add some information about the device.

tags

optional map of where you can add any information about the device in an organised manner.

position

if set to null the system will choose the last known location.

lngLat

optional longitude & latitude coordinate.

callback

optional callback for handling network related exceptions.
