Parameters

useParticleFilterAngleCorrection

determines if the map will show with the corrected angle of the device. Set to false it will be provided with the core angle of the rotation sensor, however depending on the quality of this sensor the direction of the device might drift over time.

angleCorrectionDegreePerMinute

determines how many degrees per minute the angle correction algorithm is allowed to correct. Default set to 10.
