ConfidencePositionSettings

fun ConfidencePositionSettings(confidencePositionActivated: Boolean = false, useConfidencePosition: Boolean = false, confidenceLevel: Float? = null)
Content copied to clipboard

Parameters

confidencePositionActivated

determines if the system should calculate the confidence position or not. May impact performance and battery usage

confidenceLevel

value set from 0 to 1 determining the level of confidence that the position will display in the map. This value can be configured in the CMS, setting this value here will override the value in the CMS. Set to null will default to preset level of confidence.

useConfidencePosition

determines if the map will show with the confidence position or not. Set to false it will be provided with the most weighted position and will not represent the current uncertainty regarding the location of the user.
