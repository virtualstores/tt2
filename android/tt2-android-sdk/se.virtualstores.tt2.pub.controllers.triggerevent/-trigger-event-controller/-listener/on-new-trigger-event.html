abstract fun onNewTriggerEvent(triggerEvent: TriggerEvent)
Content copied to clipboard

On new trigger event. A default TriggerEvent will be made and posted to listeners. The TriggerEvent will have some preset values depending on what type of TriggerEvent was triggered. For example. If a user entered a zone with a message connected to it, a TriggerEvent will be created and sent to listeners. The TriggerEvent will contain name equals to DefaultZoneTriggerEvent, users position, time of event and more.
