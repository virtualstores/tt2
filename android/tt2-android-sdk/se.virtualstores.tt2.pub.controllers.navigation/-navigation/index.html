abstract fun syncPosition(scanLocation: IScanLocation)

Start positioning using IScanLocation. This is the best way to start the positioning service. Scan a barcode or qr code representing a scan location that has been declared in the store settings on the CMS. Filter the active scan locations accessed by TT2.activeStore.startScanLocations() to find the scan location that matches the result from the scanner. Example code: TT2.activeStore.startScanLocations.find { location -> location.code == scanResult }?.let{ TT2.navigation.syncPosition(it) }

When the direction is uncertain, it's possible to sync position with only a point on the map. This method could be used when a user scans a product and the direction of the device is unknown. It's not recommended to use this sync position if optimal positioning is wanted.
