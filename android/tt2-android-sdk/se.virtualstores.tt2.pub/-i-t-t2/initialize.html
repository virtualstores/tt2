initialize

abstract fun initialize(context: Context, apiUrl: String, apiKey: String, clientId: Long, debugMode: Boolean = false, serviceNotificationIntent: Intent, positionKitParams: PositionKitParams = PositionKitParams(), velocityModelParams: TT2ModelParams = TT2ModelParams(), enableWorkManager: Boolean = true, enableProxy: Boolean = false, callback: (Throwable?) -> Unit)

Initialize the SDK. Call this method first before any other method to prepare the SDK for all purposes. Check the @param callback to see if everything is set.

Parameters

context

Application Context

apiUrl

Api url

apiKey

Api key

clientId

Client id

debugMode

Debug mode

serviceNotificationIntent

Service notification intent. Used to show a persistent notification to keep the SDK active on background.

positionKitParams

Params for position service. This is set to Default, don't change this unless instructed to do so.

velocityModelParams

Params for velocity model. This is set to Default, don't change this unless instructed to do so.

enableWorkManager

Params deactivating the usage of the android work manager, set to false only if your application does not support work manager.

enableProxy

Params if set to true the ML model will be downloaded via the data server as a temporary proxy solution. Don't change this unless instructed to do so.

callback

Callback contains any error during initialization if any.
