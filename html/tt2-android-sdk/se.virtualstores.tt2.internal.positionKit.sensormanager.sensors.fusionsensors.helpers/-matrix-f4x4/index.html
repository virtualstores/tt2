The Class MatrixF4x4. Internal the matrix is structured as [ x0 , y0 , z0 , w0 ] [ x1 , y1 , z1 , w1 ] [ x2 , y2 , z2 , w2 ] [ x3 , y3 , z3 , w3 ] it is recommend that when setting the matrix values individually that you use the set{x,#} methods, where 'x' is either x, y, z or w and # is either 0, 1, 2 or 3, setY1 for example. The reason you should use these functions is because it will map directly to that part of the matrix regardless of whether or not the internal matrix is column major or not. If the matrix is either or length 9 or 16 it will be able to determine if it can set the value or not. If the matrix is of size 9 but you set say w2, the value will not be set and the set method will return without any error.
